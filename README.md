# Nested React Flow Sample

React + TypeScript + Vite プロジェクトです。React Flow のノードをクリックすると、ノード内部に別の React Flow を表示してネストされたグラフを確認できます。最初にクリックされたノードは自動的に拡大され、他のノードとの差分を強調しています。

## 使い方

1. 依存関係をインストールします。
   ```bash
   npm install
   ```
2. 開発サーバーを起動します。
   ```bash
   npm run dev
   ```
3. ブラウザで表示されるフローを操作し、ノードをクリックしてネストされた構造を確認してください。

## 実装メモ

- `src/NestedFlow.tsx` にグラフ定義と、クリック時に子グラフを展開するカスタムノード (`nestable` ノード) を定義しています。
- `MiniReactFlow` コンポーネントでノード内専用の React Flow インスタンスを描画し、ズームやドラッグを無効化して親フローを邪魔しないようにしています。
- 一番最初にクリックされたノードの ID を `primaryExpandedId` として保持し、そのノードを拡大＆強調表示しています。
